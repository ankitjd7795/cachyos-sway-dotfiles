# Installation

Detailed installation instructions for CachyOS Sway Dotfiles.

## Prerequisites

- CachyOS or Arch-based Linux distribution
- `chezmoi` for dotfile management
- Git

## Quick Installation

```bash
# One-line installation
chezmoi init --apply https://github.com/ankitjd7795/cachyos-sway-dotfiles.git && ~/.local/share/chezmoi/executable_arch-setup-script.sh
```

For more details, see the [Quick Start](quick-start.md) guide.
